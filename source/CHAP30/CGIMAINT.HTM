Instructions

Enter a last name, first name and any other additional information and press the button for the action to perform. If the action is invalid then an error message will be displayed.

If you wish to inquire on a name enter the exact value from one of the following names: ):

  • William Jackson
  • Michelle Edwards
  • Angelo Lopez
  • George Dubois
Any other name which is in the contact file may also be used.

An inquire, change or delete will only succeed if the record already exists. An add will only succeed if the record does not already exist.
